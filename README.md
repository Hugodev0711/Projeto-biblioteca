# 📚 Projeto Biblioteca

Desenvolvi este projeto com o objetivo de praticar conceitos de **POO (Programação Orientada a Objetos)** e **MySQL**, simulando o funcionamento de um sistema de biblioteca.

---

## 🚀 Como executar o projeto

1. Execute o script `biblioteca.sql` no seu MySQL  
2. Configure o usuário e senha no arquivo `ConexaoDao`  
   - Padrão: usuário `root` | senha `1234`  
3. Execute o projeto  

---

## ⚙️ Funcionalidades

- Cadastro de usuários e livros  
- Listagem e busca de usuários, livros e empréstimos  
- Sistema de empréstimos com controle e validação de estoque  
- Sistema de devolução com atualização automática do estoque  

---

## 🧠 Conceitos aplicados

- Programação Orientada a Objetos (POO)  
- Integração com banco de dados (MySQL)  
- PreparedStatement e ResultSet  
- Procedures no banco de dados  
- Relacionamento entre entidades
